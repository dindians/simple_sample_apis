Reactive Relational Database Connectivity (R2DBC) project brings reactive programming APIs to relational databases.
- https://r2dbc.io/

R2DBC is an open specification and establishes a Service Provider Interface (SPI) for driver vendors to implement and clients to consume.
- https://github.com/r2dbc/r2dbc-spi/
